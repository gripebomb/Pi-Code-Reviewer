# Reference Documentation

Supporting documents loaded by the code-reviewer skill during execution.

## Files

- **`review-rubric.md`** — Detailed evaluation criteria for each of the five review categories (Code Quality, Refactoring, Documentation, Security, Test Coverage).
- **`severity-guidelines.md`** — Criteria for classifying findings as High, Medium, or Low severity, with examples.
- **`output-contract.md`** — Exact report structure and formatting rules for `.planning/REVIEW.md` and `.planning/REVIEW-TODO.md`.
- **`INSTALL.md`** — Installation and smoke-test instructions for the package.
